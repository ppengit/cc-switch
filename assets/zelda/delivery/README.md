# Zelda 图标交付说明

本目录是已整理好的交付版，默认使用 `recommended`。

## 目录结构

- `recommended/windows`
  - Windows EXE 图标
- `recommended/web`
  - favicon、PWA、apple touch icon
- `recommended/ios`
  - `app-icon-1024.png` 和 `AppIcon.appiconset`
- `recommended/android`
  - `launcher-192.png`、`launcher-512.png`、`launcher-monochrome-512.png` 以及完整 `res` 目录
- `recommended/brand`
  - 透明品牌图标与单色品牌图标
- `docs`
  - 预览图与源图说明
- `packages`
  - 已打好的压缩包

## 各平台取用

- Windows：
  - `recommended/windows/app.ico`
- Web：
  - `recommended/web/favicon.ico`
  - `recommended/web/favicon-16x16.png`
  - `recommended/web/favicon-32x32.png`
  - `recommended/web/pwa-192.png`
  - `recommended/web/pwa-512.png`
  - `recommended/web/apple-touch-icon.png`
- iOS：
  - `recommended/ios/app-icon-1024.png`
  - `recommended/ios/AppIcon.appiconset`
- Android：
  - `recommended/android/launcher-192.png`
  - `recommended/android/launcher-512.png`
  - `recommended/android/launcher-monochrome-512.png`
  - `recommended/android/res`
- 品牌展示：
  - `recommended/brand/logo-mark.png`
  - `recommended/brand/logo-mark-512.png`
  - `recommended/brand/logo-mark-256.png`
  - `recommended/brand/logo-mark-mono.png`

## 说明

- 这套资源是单方案交付，没有再拆分多个风格分支。
- 品牌目录提供的是图标 mark，不包含横版文字 logo。
