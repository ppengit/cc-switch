# 后端接口与数据库变更说明

## 1. 数据库变更

新增会话标题映射表（本地映射）：

- 主键建议：`provider_id + session_id + source_path`
- 字段：`custom_title`, `updated_at`

用途：允许应用内重命名会话，不回写原应用。

## 2. 会话命令扩展

- `list_sessions`：叠加映射标题。
- `set_session_title_mapping`：设置/更新映射。
- `clear_session_title_mapping`：删除映射。
- `get_recent_sessions`：可选后端提供，按应用返回最近会话。

## 3. 配置命令扩展

- `get_app_config_files(app)`：列出可编辑配置文件。
- `read_app_config_file(app, path)`：读取文件内容。
- `write_app_config_file(app, path, content)`：写回文件。
- `get_app_config_template(app)` / `set_app_config_template(app, template)`。

## 4. 使用统计命令扩展

请求日志查询补充字段：

- `session_title`
- `project_path`
- `metadata`（详情用）

## 5. 安全与校验

- 文件读写限制在应用配置目录内。
- 写入前校验目标路径规范化后仍在允许目录。
- 命令异常统一返回可读错误文本。
