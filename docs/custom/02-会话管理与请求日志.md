# 会话管理与请求日志改造

## 1. 会话管理改造

- 在“删除会话”前增加“一键恢复”按钮。
- 在“会话名称/标题”后增加“编辑标题”按钮。
- 标题编辑写入本地数据库映射，不改原会话文件。
- 会话恢复调用统一终端启动能力，按用户终端偏好执行。

## 2. 最近会话入口

- 在供应商列表工具栏提供最近会话下拉。
- 来源使用会话管理同一数据源，按应用筛选并按最近活跃排序。
- 支持点击即恢复并带项目路径 cwd。

## 3. 请求日志改造

- 列表新增：会话标题（缩略）/项目路径。
- 支持双击行打开请求元数据详情。
- 详情面板显示 request metadata（若存在）与扩展上下文信息。

## 4. 兼容策略

- 老日志记录无会话标题时使用回退显示（会话 ID / 空）。
- 元数据为空时详情面板展示“无元数据”。
